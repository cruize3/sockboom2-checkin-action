## 使用方法

在 `Settings` -> `Secrets and variables` -> `Actions` 新建下列 `New repository secret`

- TOKEN
  - 获取在GitHub 右上角头像 -> `Setting` -> `Developer settings` -> `Personal access tokens` -> `Tokens(classic)` 中获取
- EMAIL
  - SockBoom邮箱
- PASSWORD
  - SockBoom密码
- SUFFIX
  - SockBoom域名后缀


## 启用

`Action` -> `SockBoom Checkin Action` -> `Enable Workflow`
